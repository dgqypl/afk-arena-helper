# 剑与远征 - 副本微调助手

## 项目简介

这是一个帮助《剑与远征》玩家优化"诅咒梦境"和"梦魇回廊"副本阵容的辅助工具。

### 核心问题
使用 B 站 UP 主（如[狐小鱼](https://space.bilibili.com/64090133)）或 Reddit 社区的参考阵容时，经常出现：
- 伤害远低于参考值
- 用时远超参考值
- 需要通过"微调"（脱装备/家具等）来达到预期效果

### 解决方案
通过建立**战斗现象诊断 + 经验规则库**，快速给出微调建议，避免盲目摸索。

### 对话规范
- **英雄称呼**：与用户对话时**必须使用中文简称**（如"秋神"、"SP女骑"、"棋神"），不要使用中文全名（如"奥莱妮娅"、"爱丝特瑞达"、"卢迪亚洛"）。中文简称见 `英雄库.md` 英雄总表的"中文简称"列。
- **副本简称**：诅咒梦境 = "诅咒"，梦魇回廊 = "竞速"，猎宝角逐 = "TS"
- **副本结构**：诅咒和竞速各有 **6 队**。1-5 队各自对阵不同阵营的敌方英雄（如一队对阵 5 个耀光英雄），第 6 队打 BOSS
- **术语对照**：Haste = 急速，ATK = 攻击力，DEF = 防御，HP = 生命值，ERR = 回能效率，Erosion = 腐蚀，PP = 破防，MP = 破魔，CDA = 暴伤增幅，IS = 洞察，TY = 韧性，PRF = 精通，b = billion（十亿，用于伤害数值，如 39.3b = 393亿）
- **未知英雄**：如果用户提到的阵容中有英雄库里没有的英雄，**必须主动询问用户**该英雄的信息，绝不自行猜测
- **禁止猜测**：遇到任何不理解的游戏术语、机制或概念时，**必须反问用户**，绝不自行猜测或编造
- **主动更新文档**：当用户在对话中提到任何新知识（新英雄、藏品别名、游戏机制、术语等），**必须主动更新到对应文档**，不要等用户要求。对应关系：新英雄→`英雄库.md`，藏品别名/新藏品→`英雄库.md`+`藏品技能效果.md`，游戏基础机制→`CLAUDE.md`，诊断规则/微调经验→`诊断规则库.md`，新参考资源→`CLAUDE.md`
- **不要重复问已提供的信息**：用户在对话中已经提供过的数据（如英雄练度），不要再次询问，自己回头查找

### 装备基础知识
- **品质**（从高到低）：传说(红) > 史诗(金) > 精英+(紫) > 精英(紫)
- **等级**：T1/T2 通用装备（可升 1-5 星，T2 五星可升 T3），T3/T4 英雄专属装备（无星级）
- **种族属性**：本族英雄穿本族装备有额外加成。穿其他种族装备属性加成更少
- **微调常用手段**：换低级鞋子（如 T1 5星、史诗金鞋子）来精确调控急速

### 种族克制关系
- 耀光 克 蛮血
- 蛮血 克 绿裔
- 绿裔 克 亡灵
- 亡灵 克 耀光
- 半神 与 恶魔 互相克制
- 虚空：不克别的种族，也不被别的种族克制
- 龙族：与虚空相同，无克制关系

### 练度标记法
玩家用 **5位数字** 表示英雄练度，拆分为 `XX-Y-ZZ`：
- **前2位**：专属等级（SI）
- **中间1位**：红家具数量（F）
- **后2位**：刻印等级（E）
- 示例：`30960` = 30专属 / 9红家具 / 60刻印，`20930` = 20专属 / 9红家具 / 30刻印

## 工作流程

### 1️⃣ 遇到问题时
在 `本周副本.md` 中记录：
- 使用的阵容（英雄 + 站位 + 藏品 + 宠物）
- 参考用时/伤害 vs 实际用时/伤害
- 战斗中观察到的现象

### 2️⃣ Claude 诊断
- 读取 `诊断规则库.md` 中的已知规则（含 Reddit 社区攻略总结）
- 读取 `历史案例.md` 中的相似案例
- 读取 `英雄库.md` 了解英雄机制和藏品/词条数据
- 读取 `藏品技能效果.md` 了解藏品各等级技能
- 读取 `宠物库.md` 了解宠物技能和选择
- 给出微调建议优先级

### 3️⃣ 尝试并反馈
- 按优先级依次尝试微调方案
- 记录每次尝试的结果
- 成功后更新到 `历史案例.md`
- 如果发现新规律，更新 `诊断规则库.md`

## 使用说明

### 初次使用
1. 在 `英雄库.md` 中添加你常用的英雄信息
2. 查看 `诊断规则库.md` 了解已有规则

### 每周打本
1. 更新 `本周副本.md` 的副本信息
2. 遇到问题时按模板记录
3. 与 Claude 对话获取微调建议
4. 成功后记录到 `历史案例.md`

### 持续优化
- 案例越多，诊断越准确
- 规则库会不断完善
- 逐步补充英雄机制信息
