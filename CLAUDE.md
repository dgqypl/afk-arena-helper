# 剑与远征 - 副本微调助手

## 项目简介

这是一个帮助《剑与远征》玩家优化"诅咒梦境"和"梦魇回廊"副本阵容的辅助工具。

### 核心问题
使用 B 站 UP 主（如[狐小鱼](https://space.bilibili.com/64090133)）或 Reddit 社区的参考阵容时，经常出现：
- 伤害远低于参考值
- 用时远超参考值
- 需要通过"微调"（脱装备/家具等）来达到预期效果

### 解决方案
通过建立**战斗现象诊断 + 经验规则库**，快速给出微调建议，避免盲目摸索。

### 对话规范
- **英雄称呼**：与用户对话时**必须使用中文简称**（如"秋神"、"SP女骑"、"棋神"），不要使用中文全名（如"奥莱妮娅"、"爱丝特瑞达"、"卢迪亚洛"）。中文简称见 `英雄库.md` 英雄总表的"中文简称"列。
- **副本简称**：诅咒梦境 = "诅咒"，梦魇回廊 = "竞速"
- **术语对照**：Haste = 急速，ATK = 攻击力，DEF = 防御，HP = 生命值，ERR = 回能效率，Erosion = 腐蚀，PP = 破防，MP = 破魔，CDA = 暴伤增幅，IS = 洞察，TY = 韧性，PRF = 精通
- **未知英雄**：如果用户提到的阵容中有英雄库里没有的英雄，**必须主动询问用户**该英雄的信息，绝不自行猜测

### 装备基础知识
- **品质**（从高到低）：传说(红) > 史诗(金) > 精英+(紫) > 精英(紫)
- **等级**：T1/T2 通用装备（可升 1-5 星，T2 五星可升 T3），T3/T4 英雄专属装备（无星级）
- **种族属性**：本族英雄穿本族装备有额外加成。穿其他种族装备属性加成更少
- **微调常用手段**：换低级鞋子（如 T1 5星、史诗金鞋子）来精确调控急速

## 工作流程

### 1️⃣ 遇到问题时
在 `本周副本.md` 中记录：
- 使用的阵容（英雄 + 站位 + 藏品 + 宠物）
- 参考用时/伤害 vs 实际用时/伤害
- 战斗中观察到的现象

### 2️⃣ Claude 诊断
- 读取 `诊断规则库.md` 中的已知规则（含 Reddit 社区攻略总结）
- 读取 `历史案例.md` 中的相似案例
- 读取 `英雄库.md` 了解英雄机制和藏品/词条数据
- 读取 `藏品技能效果.md` 了解藏品各等级技能
- 读取 `宠物库.md` 了解宠物技能和选择
- 给出微调建议优先级

### 3️⃣ 尝试并反馈
- 按优先级依次尝试微调方案
- 记录每次尝试的结果
- 成功后更新到 `历史案例.md`
- 如果发现新规律，更新 `诊断规则库.md`

## 使用说明

### 初次使用
1. 在 `英雄库.md` 中添加你常用的英雄信息
2. 查看 `诊断规则库.md` 了解已有规则

### 每周打本
1. 更新 `本周副本.md` 的副本信息
2. 遇到问题时按模板记录
3. 与 Claude 对话获取微调建议
4. 成功后记录到 `历史案例.md`

### 持续优化
- 案例越多，诊断越准确
- 规则库会不断完善
- 逐步补充英雄机制信息
