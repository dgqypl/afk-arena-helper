# AFK Arena 副本微调助手

一个基于 [Claude Code](https://claude.ai/download) 的《剑与远征》副本阵容微调工具。通过维护本地知识库（英雄、藏品、宠物、诊断规则、历史案例），让 AI 帮你快速诊断"诅咒梦境"和"梦魇回廊"中阵容表现不佳的原因，并给出微调建议。

## 它能解决什么问题？

抄 B 站 UP 主或 Reddit 社区的参考阵容时，经常遇到：
- 用时远超参考值（比如参考 6.97s，你打出 16.6s）
- 伤害远低于参考值
- 不知道该调什么、怎么调

原因通常不是你练度不够，而是隐藏的属性差异（刻印、家具、专武等级等）导致**技能释放节奏不同步**。这个工具帮你系统化地定位问题并找到微调方案（比如脱某个英雄的武器、换低级鞋子调急速等）。

## 工作原理

```
你描述问题 → Claude 读取知识库 → 给出微调建议 → 你尝试并反馈 → 更新知识库
```

项目中的 Markdown 文件就是知识库，Claude 通过读取它们来理解你的阵容和已有经验：

| 文件 | 用途 |
|------|------|
| `英雄库.md` | 你常用的英雄信息（名称、定位、种族、藏品、机制备注） |
| `宠物库.md` | 宠物技能详细数据和选择指南 |
| `藏品技能效果.md` | 17 个藏品的各等级技能效果 |
| `诊断规则库.md` | 从实战和社区攻略总结的诊断规则（急速调控、ATK 排序、装备脱卸等） |
| `历史案例.md` | 完整的微调过程记录（问题 → 尝试 → 解决方案 → 经验总结） |
| `本周副本.md` | 当前周的副本阵容规划和待解决问题 |
| `CLAUDE.md` | Claude 的行为指引（对话规范、术语对照、工作流程） |

## 快速开始

### 前提条件

- 安装 [Claude Code](https://claude.ai/download)（Anthropic 官方 CLI 工具）
- 一个 Anthropic API 账号或 Claude Pro/Max 订阅

### 1. Fork 本仓库

点击 GitHub 页面右上角的 **Fork** 按钮，将仓库复制到你自己的账号下，然后克隆到本地：

```bash
git clone https://github.com/<你的用户名>/afk-arena-helper.git
cd afk-arena-helper
```

### 2. 填入你的英雄数据

当前的英雄列表可以直接使用。如果你需要增删英雄，直接告诉 Claude 即可，比如：

> 我的英雄库里还需要加上一拳超人（Saitama），他是虚空战士。

在使用过程中发现的英雄机制（比如"秋神下武器可降低回能速度"），Claude 也会自动帮你记录到 `英雄库.md` 的"英雄机制备注"部分。所有知识库文件的维护都通过和 Claude 对话完成，你不需要手动编辑这些 Markdown 文件。

### 3. 开始使用

在项目目录下启动 Claude Code：

```bash
claude
```

然后直接用中文描述你的问题，例如：

> 我竞速二队用的 SP女骑/SP天罚/秋神/棋神/戒律龙，参考用时 6.97s，但我打出来 16.6s，没有人死，就是慢。怎么调？

Claude 会自动读取知识库中的规则和历史案例，给你诊断和微调建议。

### 4. 记录结果

微调成功后，告诉 Claude 结果，它会帮你：
- 把成功案例写入 `历史案例.md`
- 如果发现了新规律，更新 `诊断规则库.md`

**案例越多，后续诊断越精准。**

## 每周使用流程

```
1. 周一更新 本周副本.md（BOSS 信息、参考阵容）
2. 遇到问题时和 Claude 对话，描述现象
3. 按 Claude 的建议依次微调
4. 成功后让 Claude 记录到 历史案例.md
5. 发现新规律时让 Claude 更新 诊断规则库.md
```

## 知识库说明

### 诊断规则库

已收录的规则包括：

- **NC-001** 速通阵容技能轴不同步 — 脱武器调回能
- **NC-002** Haste（急速）调控 — 脱/换鞋子精确调控
- **NC-003** ATK 排序要求 — 某些英雄需要成为全队最高/最低 ATK
- **NC-004** 装备脱卸速查表 — 常见英雄的脱装备方案
- **CR-001** 核心英雄属性需求 — 诅咒梦境关键属性
- **CR-002** 藏品搭配要点 — 高频推荐的藏品+词条组合
- **COMMON-001~005** 通用规则（生存、伤害、装备、换位、最低投资）

规则来源包括 Reddit 社区攻略（mugen_tsukuyomi 等作者）和个人实战经验。

### 宠物库

收录了 24 个宠物的完整技能数据，包括各等级解锁效果、团队属性加成和关键机制索引。

### 藏品技能效果

收录了 17 个藏品的一星/二星/三星技能效果，附带诊断要点标注。

## 如何贡献 / 自定义

这个仓库的核心价值在于**你自己积累的数据**。Fork 之后你可以：

- 在 `英雄库.md` 中添加/删除你常用的英雄
- 在 `历史案例.md` 中记录你自己的微调经验
- 在 `诊断规则库.md` 中添加你发现的新规律
- 修改 `CLAUDE.md` 来调整 Claude 的对话风格和工作流程
- 在 `本周副本.md` 中维护你每周的阵容记录

如果你发现了对其他玩家也有帮助的通用规则，欢迎提 PR 回主仓库。

## 常见问题

**Q: 需要付费吗？**
A: Claude Code 本身需要 Anthropic API 额度或 Claude Pro/Max 订阅。本仓库是免费的。

**Q: 数据会上传到服务器吗？**
A: 所有知识库文件都存储在你本地。与 Claude 的对话内容会发送到 Anthropic API，但 Markdown 文件本身不会被自动上传到任何地方。

**Q: 英雄库里没有我用的英雄怎么办？**
A: 直接在 `英雄库.md` 中添加即可。和 Claude 对话时提到未知英雄，它也会主动询问你该英雄的信息。
