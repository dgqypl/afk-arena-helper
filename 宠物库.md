### 1. 海豹 / Slumber Seal

**定位**：最强输出宠物，可单独carry战斗
**关键机制**：增伤、泡泡控制、百分比最大生命值伤害

| 等级 | 技能名/类型 | 效果描述 |
|------|-----------|---------|
| Lvl 0 | 被动：Aquatic Spirit | 每1.5秒生成1个水灵攻击血量最低的敌人，造成220%攻击力的宠物伤害。每当友方英雄受到或抵抗控制效果时，在海豹周围掉落1个水灵（每6秒最多5个）。可回收水灵，永久增加水灵伤害12.5%，最多叠加至200% |
| Lvl 3 | 被动增强 | 每次回收水灵时，攻击力永久增加触发英雄当前攻击力的15%，获得的属性不超过所有友方英雄初始攻击力总和的60%。团队属性加成：HP 2.19%, DEF 5.93%, ATK 4.8% |
| Lvl 6 | 主动：Ocean Song | 海豹移至战场中央，掀起3波浪潮攻击敌人。每次被浪潮击中的敌人受到300%攻击力的宠物伤害并被击退。友方英雄受到或抵抗控制效果时，海豹获得额外宠物能量 |
| Lvl 9 | 主动增强 | 当友方英雄5秒内未受到或抵抗控制效果时，海豹获得额外宠物能量并生成1个水灵 |
| Lvl 12 | 主动增强 | 最后一波浪潮击中的敌人被困在泡泡中3秒。被困敌人受到的伤害增加40%，宠物伤害额外增加40% |
| Lvl 15 | 增强 | （具体描述未在网页中完整公开） |
| Lvl 18 | 主动增强 | 每当敌人被困在泡泡中时，海豹生成额外1个水灵攻击该敌人，造成敌人最大生命值10%的伤害（上限为600%攻击力） |

### 2. 猫头鹰 / Flutterplume Owl

**定位**：最强辅助型宠物，时间操控
**关键机制**：延长友方buff持续时间、缩短敌方buff时间、免控、冻结、加速

| 等级 | 技能名/类型 | 效果描述 |
|------|-----------|---------|
| Lvl 0 | 主动：Chrono Control | 在英雄最密集的区域生成时间场，对区域内敌人造成330%攻击力的伤害，改变区域时间流速8秒。区域内友方英雄大部分buff持续时间延长20%，敌方英雄大部分buff持续时间缩短20%。友方英雄对敌方造成伤害时，猫头鹰获得额外宠物能量 |
| Lvl 3 | 被动 | 猫头鹰持续侦察战场，提升所有友方英雄洞察力（Insight）10点，增加其造成的伤害18%。团队属性加成：HP 4.38%, ATK 3.44%, DEF 5.04% |
| Lvl 6 | 被动：CC免疫 | 定期使累计伤害最高的友方英雄短暂免疫控制效果，并在接下来5秒增加其坚韧（Tenacity）30点 |
| Lvl 9 | 时间场增强 | 友方英雄在时间场中停留3秒后，短暂免疫伤害和控制效果（触发次数有限）。团队属性加成：HP 5.04%, DEF 5.79%, ATK 3.96% |
| Lvl 12 | 时间场增强 | 友方buff延长比例提升至30%，敌方buff缩短比例提升至30%。敌方英雄在时间场中停留3秒后被冻结2秒（每个时间场内每个敌人只触发1次，对Boss无效） |
| Lvl 18 | 时间场增强 | 时间场持续时间延长至10秒。区域内友方英雄加速（Haste）额外增加40点，敌方减速40点。猫头鹰使用Chrono Control 3次后，战斗剩余时间延长5秒 |

### 3. 熊猫 / Pandy Dumpty

**定位**：暴击增强型宠物
**关键机制**：暴击率、暴击伤害放大、晕眩、联合伤害

| 等级 | 技能名/类型 | 效果描述 |
|------|-----------|---------|
| Lvl 0 | 主动 | 熊猫滚入战场，在竹林覆盖区域内为友方英雄提供暴击率加成 |
| Lvl 3 | 被动 | 砸击对一定范围内的敌人造成2秒眩晕。随着熊猫升至更高位置，后续砸击的眩晕效果延长0.5秒 |
| Lvl 15 | 被动 | 竹林覆盖区域内的友方英雄造成伤害时，熊猫对目标造成该英雄200%攻击力 + 熊猫200%攻击力的宠物伤害。每个友方每1秒最多触发10次 |
| Lvl 18 | 被动 | 当友方英雄造成伤害且其暴击率超过目标暴击格挡率100点以上时，每多1点增加暴击伤害放大2。竹林覆盖区域内英雄的暴击伤害放大可突破暴击伤害上限。如果友方造成暴击伤害，熊猫的额外伤害也会暴击 |

团队属性加成：HP 3.95%, DEF 7.11%, ATK 4.8%

### 4. 幽灵 / Treatspector

**定位**：防致命伤害 + 属性削弱型
**关键机制**：防致死（Lvl 18核心）、攻击力buff、锁链减属性、恐惧

| 等级 | 技能名/类型 | 效果描述 |
|------|-----------|---------|
| Lvl 0 | 被动+主动：Candy Booster | 被动：在战场上方漂浮，向附近友方英雄投掷糖果，增加其攻击力40%持续5秒。主动：飞向敌人最密集的区域，发射光束对路径上的敌人造成150%攻击力的宠物伤害。对敌人施加Lockdown效果，每秒失去5%生命值（每次不超过400%攻击力），持续5秒。敌人移动时，Treatspector获得额外宠物能量 |
| Lvl 3 | Sweet Treat | 每投掷3颗糖果后，下一次投掷附带1颗Sweet Treat，对范围内敌人造成150%攻击力的宠物伤害并使其恐惧（Terrify），逃散3秒 |
| Lvl 6 | Lockdown增强 | 被锁链束缚的敌人每秒失去5%攻击力和5%防御力（上限45%），移动越多失去越多。效果结束后恢复 |
| Lvl 9 | 属性吸收 | 吸收敌人属性并附加到糖果增强器上，提升友方英雄对应属性最多40% |
| Lvl 15 | Stop-and-Go | 战斗开始时，在战场中央释放Stop-and-Go，不消耗宠物能量。糖果增强器buff范围扩大，持续时间从5秒增加至10秒 |
| Lvl 18 | Soul Flame（核心技能） | 当友方英雄受到致命伤害时，Treatspector牺牲灵魂之火，将英雄恢复至过去4秒的最佳状态。每场战斗可触发2次 |

团队属性加成：HP 4.59%, DEF 10.67%, ATK 3.36%

**定位**：收集负面能量增强伤害 + 压制敌方输出
**关键机制**：负面能量收集、增伤、忧郁debuff压制、传说（Fabled）宠物

| 等级 | 技能名/类型 | 效果描述 |
|------|-----------|---------|
| 全技能 | Fabled Beast | 传说宠物，从战场收集负面能量（Negative Energy）来增强自身和友方英雄的伤害输出，同时向敌人传播忧郁（Melancholy）来压制敌方攻击。推荐投资：F2P/小R练至Lvl 21，大R练至Lvl 27。注意：不应在普通宠物共鸣达到Lvl 18之前投资传说宠物 |

### 6. 巧虎 / Talismane

**定位**：单体buff + 致命防护
**关键机制**：跟随英雄增强、鞭炮致盲、恐惧、长命锁防致死

| 等级 | 技能名/类型 | 效果描述 |
|------|-----------|---------|
| Lvl 0 | 被动 | 跟随一名英雄并提供被动属性加成 |
| Lvl 3 | A Blessed Year | 跟随的英雄首次使用大招时，巧虎立即使用1次技能，不消耗宠物能量 |
| Lvl 6 | 鞭炮致盲 | 使用技能后，巧虎在返回途中沿路投掷鞭炮，对敌人造成宠物伤害并致盲（Blind） |
| Lvl 9 | 鞭炮增强 | 鞭炮伤害进一步提升，亡灵族（Graveborn）和恶魔族（Hypogean）英雄额外被恐惧3秒。团队属性加成：HP 4.28%, DEF 8.52%, ATK 3.77% |
| Lvl 12 | 长命锁 Longevity Lock | 每10秒给跟随的英雄一个长命锁。当英雄受到超过一定比例最大生命值的单次伤害或致命伤害时，长命锁抵消该伤害并消失。持有长命锁的英雄受到的伤害降低、造成的伤害提升 |
| Lvl 18 | 长命锁扩展 | 长命锁额外给予累计伤害最高的另一名友方英雄。如果有更多目标需分配，优先给未持有过长命锁的英雄。跟随英雄持有长命锁期间，闪避和魔法压制提升至3倍。团队属性加成：HP 4.47%, DEF 8.89%, ATK 3.94% |

| 等级 | 技能名/类型 | 效果描述 |
|------|-----------|---------|

### 8. 雪花 / Ice Crown

**定位**：强力冻结控制 + 护盾吸伤
**关键机制**：冻结（核心）、霜盾免召唤物伤害、暴击削弱

| 等级 | 技能名/类型 | 效果描述 |
|------|-----------|---------|
| Lvl 0 | 主动：Glacial Elegy | 战斗开始时在友方半场冰封休眠。主动释放时破冰而出，降下3波冰晶碎片，每波对所有敌人造成250%攻击力的宠物伤害。为友方英雄赋予霜盾（Frost Shield）持续8秒，驱散大部分debuff。霜盾使英雄免疫敌方召唤物和宠物的伤害，30%来自敌方英雄的伤害由Ice Crown吸收（上限最大生命值25%，主动释放时重置） |
| Lvl 3 | 被动 | 战斗开始时，在周围区域覆盖霜冻，持续一段时间减少受影响敌人的加速（Haste）60点。团队属性加成：HP 3.29%, DEF 5.93%, ATK 4% |
| Lvl 6 | 冻结 | 主动释放时，冻结生命值比例高于50%的敌人5秒。被冻结的敌人每受到1次伤害，冻结时间延长0.5秒（最多延长3秒）。若敌人抵抗或免疫冻结，其坚韧永久降低20 |
| Lvl 9 | 霜盾增强 | 霜盾吸伤比例提升至45%。下次主动释放时，造成已吸收伤害15%的额外宠物伤害（上限800%攻击力），吸收量重置 |
| Lvl 12 | 开场冻结（核心技能） | 战斗开始时休眠状态下吸收友方70%的受伤（上限最大生命值50%）。当吸收量达到上限或开战6秒后，立即破冰并主动释放Glacial Elegy，不消耗宠物能量 |
| Lvl 18 | 暴击削弱 | 被冻结的敌人失去70点暴击格挡率（Crit Block Rate）和15点暴击伤害抗性（Crit Damage Resistance） |

### 11. 宝箱 / Trickery Chest

**定位**：能量操控 + 防致死型
**关键机制**：废大招、吸能量、防致死（Lvl 18）

| 等级 | 技能名/类型 | 效果描述 |
