## 藏品 (Artifacts)

藏品以英雄的中文简称命名（但藏品对应的英雄不一定在常用英雄列表中）。
藏品技能效果参考：https://afk.guide/ghoulish-gallery-and-collections-guide/

### 藏品等级机制

每个藏品有三个等级：

| 等级 | afk.guide 叫法 | 游戏内显示 | 说明 |
|------|---------------|-----------|------|
| 一星 | Level 0 | 1 颗星 | 默认等级 |
| 二星 | Level 1 | 2 颗星 | 镶嵌 1 个"技能等级"词条 |
| 三星 | Level 2 | 3 颗星 | 镶嵌 2 个"技能等级"词条（满级） |

- 藏品通过选择纹签打造，随机掉落词条后选择镶嵌
- **"技能等级"词条**是提升藏品等级的关键，最多镶嵌 2 个
- 藏品等级越高，技能效果越强

> **诊断相关**：藏品等级差异可能导致同阵容表现不同。如果参考阵容使用三星藏品而你是一星，技能效果差距可能很大。

### 词条分类速查

- **攻击类**：攻击, 全队攻击, 洞察, 破魔, 破防, 暴击率, 暴伤增幅, 腐蚀, 精通, 全队腐蚀
- **生存类**：生命, 全队生命, 防御, 全队防御, 伤害抗性, 韧性, 化解, 暴伤耐性, 暴击招架, 护盾效果, 吸血等级
- **辅助类**：治疗效果, 缓和, 全队缓和
- **特殊类**：回能效率, 技能等级

> **注意**：回能效率 (ERR) 词条与微调诊断密切相关——直接影响技能释放速度。

### 词条品阶

每条词条有自己的品阶（颜色），品阶越高属性加成越大：

| 品阶 | 颜色 | 说明 |
|------|------|------|
| 红色+ | 红 | 极少数才能打造出的最高品阶 |
| 红色 | 红 | 主流品阶，一般说"几攻击、几破防"等默认指红色 |
| 橙色 | 橙 | 关键词条勉强能用，但不如红色 |
| 紫色 | 紫 | 属性加成远低于橙色，基本没用 |
| 蓝色 | 蓝 | 属性加成远低于橙色，基本没用 |
| 绿色 | 绿 | 属性加成远低于橙色，基本没用 |

> **实用标准**：合格藏品的关键词条至少要红色品阶，橙色只能算凑合用。

## 英雄机制备注

> 以下信息通过实战案例逐步积累，初始为空。

### 奥莱妮娅 / Aurelia (秋神) - 法师
- 高频输出型英雄
- 回能速度较快
- 容易出现"技能轴过快"问题
- **微调经验**：下武器可降低回能速度，适用于需要延缓技能释放的场景（见[案例001](历史案例.md#案例001)）

### 阿塔利娅 / Athalia (SP天罚) - 战士
- **伤害类型**：物理为主（3红激活后普攻附加魔法伤害）
- **核心机制**：切攻击力最高的敌人，Ripping Slash期间无敌；SI+20对目标已损失生命值有额外伤害（10%，上限400%攻击力）
- **回能方式**：无特殊回能，主要靠普攻+受击（能量积累较慢）
- **暴伤增幅(CDA)价值**：终极和裂斩为爆发技能，CDA直接提升爆发伤害上限；E60每次普攻削减敌方20%能量
- **微调经验**：见诊断规则库 CR-001（需CDA+ATK）

### 卢迪亚洛 / Lysander (棋神) - 游侠
- **伤害类型**：物理
- **核心机制**：位置决定功能——前排=兵（切入后变骑士），后排中=王（全队减伤+给最近2友方激励），后排侧=车（打最远目标）
- **首次终极免费**：开战自动释放一次终极，不消耗能量
- **激励buff（王状态）**：回能效率+20、韧性+50、闪避+20%、物穿+魔穿+75%；SI+40全队生效
- **破防(PP)价值**：骑士状态永久降低目标10%防御（最多6层），配合破防进一步穿透物理防御
- **微调经验**：见诊断规则库 CR-001（需PP）

### 莱欧斯 / Laios (狗男) - 战士
- **伤害类型**：物理
- **核心机制**：SI+40全队+75%伤害（普攻叠信赖层，满15层激活）；料理每8秒给全队+50%攻击力+每次治疗+20能量（最多10次/餐）
- **必杀技限制**：释放期间**无法恢复能量**，但免疫控制+减伤80%，可安全站场
- **三红家具**：每秒回复6%最大生命值，每多吃过一种食材额外+3%/秒（生存能力大幅提升）
- **九红家具**：每多吃过一种食材额外+20%攻击力和防御力（食材种类越多输出越高）
- **急速敏感性**：急速影响普攻频率（即信赖叠层速度），从而影响满层触发全队+75%伤害的时机；22与28对应不同阵容下与队伍技能轴的同步节奏，具体区别来自Reddit攻略，待实战验证
- **玛露联动**：与玛露同队时，料理每次分发额外给玛露+50能量；玛露获得"忠犬"+40%攻击力
- **微调经验**：见诊断规则库 NC-002（急速 22/28）

### 拉瓦图恩 / Lavatune (吉他) - 辅助
- **核心定位**：能量电池 + 急速加速器 + 终极增伤
- **能量恢复机制**：Torrent Metal给最低能量友方+100能量；终极期间友方在舞台释放终极→全队+100能量；141级自身每次友方终极+100能量
- **急速加速机制**：开场表演15秒，每次友方释放终极+8急速（E60上限80），表演结束后每秒衰减1点
- **终极增伤**：友方在舞台时终极伤害+60%（9红提升至+100%）
- **急速阈值重要性**：急速衰减导致最优窗口会动态变化，需配合队伍技能频率精确管理

### 托安 / Thoran (SP骨王) - 坦克
- **伤害类型**：物理
- **复活机制**：死亡消耗1把暴风之刃复活，SI+30初始5把（最多4次复活）；复活时造成300%攻击力+3秒眩晕
- **伤害公式（越死越强）**：持有暴风之刃时持续失血，**已损失生命值×1% = 额外攻击力，上限为基础攻击力×10倍**
- **E60回能**：每失去5%最大生命值获得100能量（持续失血→极快回能）
- **雷鸣debuff**：每层+20%受伤，最多5层（=+100%受伤），可叠在BOSS身上

### 维露菲拉 / Velufira (黑龙) - 辅助
- **伤害类型**：龙语魔法伤害
- **破魔(MP)价值**：自身造成魔法/龙语伤害，破魔提升自身伤害穿透（增伤而非生存）
- **结界debuff**：减少结界内敌人暴击率20点、削减魔穿和洞察、全场伤害-35%
- **阻止终极**：龙语之怒持续期间（8秒）敌人**无法使用终极技能**，是高价值控制效果
- **3红能量削减**：攻击护甲的敌人每秒失去80能量+伤害-40%
- **9红增益**：有护甲的友方+35%伤害

### 艾尔西娅 / Elthara (火龙) - 法师
- **伤害类型**：龙语伤害
- **核心输出机制**：终极将最高伤害敌人变为"烈焰仿身"，仿身每0.5秒AOE生命流失（370%攻击力），到期爆炸（440%攻击力龙语）；对BOSS在附近生成仿身持续输出
- **SI+40关键**：每点腐蚀+2龙语增幅（上限100），每点魔穿+5%龙语伤害（上限150%）→ 高腐蚀+魔穿配置对火龙输出倍增
- **全队焰核护盾**：SI开局给全队护盾，累积受伤超15%时激活，SI+30联动仿身额外输出
- **微调经验**：见[案例005](历史案例.md#案例005)（竞速尾队主要输出位）

### 塔莉玛尔 / Talimar (钓鱼妹) - 辅助
- **伤害类型**：智力型
- **BOSS增伤机制**：终极对不可移动BOSS附加"受伤+35%"debuff，最多叠2次（=+70%），每次终极刷新覆盖
- **鱼饵DoT**：普攻叠加鱼饵层数（最多6层），每层每2秒+50%攻击力伤害
- **E60加成**：被撕裂的敌人至少多受35%伤害，最高+100%
- **穿装vs脱装影响原因**：急速影响终极释放频率，从而影响+70%受伤debuff的覆盖时长；鱼饵叠层速度也受急速影响
- **9红**：最高攻击力友方也获得鱼饵和渔具升级效果

### 利贝塔 / Liberta (圣子) - 辅助
- **定位**：能量操控 + 持续治疗/伤害
- **球的目标选择**：庇佑珠→**当前HP最低的友方**（除自身），惩击珠→**当前HP最高的敌方**；必杀技链接**HP最低的友方**+HP最高的敌方
- **惩击珠4级关键**：移除时持有者受伤害永久+16%，最多叠5次（=+80%），是关键增益积累
- **SI能量操控**：SI+20每球±150能量（给友方+150，削减敌方-150）
- **与元歌（阿德里安&伊莉斯）联动**：通过脱元歌胸甲+头盔让元歌HP最低，确保庇佑珠和必杀技持续锁定元歌；链接期间庇佑珠的治疗→100%转化为被链接敌方的生命流失，元歌成为"治疗转伤害中转站"
- **秩序重归4级**：移除庇佑珠生成惩击珠，移除惩击珠生成庇佑珠，双向循环不断积累惩击层数
- **微调经验**：见诊断规则库 NC-003（脱元歌胸甲+头盔控制HP排序）

### 黛弥亚 / Daemia (夏神) - 法师
- **伤害类型**：魔法
- **花荫核心效果**：全场伤害+100%持续10秒，阻止所有治疗，期间不会直接致死（BOSS除外）
- **全队攻击buff依赖原因**：花荫期间花开之日给有"花露蜜"的友方+200能量，依赖队友高频释放终极来触发连锁；酷热风受伤叠层（9红上限+50%受伤）也需要队友高频攻击快速叠满
- **永久攻击力积累**：SI+10/+20积累4/6次花露蜜后永久+30%/+50%攻击力（自身积累）
- **酷热风 / 烈日双buff**：受伤递增（攻方buff）+ 输出递减（守方buff），形成攻防双重优势
- **161级能量转化**：多余能量→暴伤增幅（上限+50）

### 玛露西尔 / Marcille (玛露) - 法师
- **伤害类型**：魔法（超50 Arcana时无视防御）
- **Arcana机制**：开局100点（上限100），超过50时普攻**无视防御**（350%攻击力AOE）
- **最高战斗力需求原因**：竞速R2与时钟同队——时钟三红家具"时间管理者"：必杀技全场时间停止期间，**战斗力最高的友军**可如常行动（九红额外+35急速）；玛露保持战斗力第一，时间停止时独自对被冻结敌人输出
- **黑魔法印记**：改写召唤物死亡时叠加，每层+60-80%伤害（最多3层），是关键爆发增益
- **3红保命**：单次致命伤→放逐3秒→复活（回复500%攻击力生命值）
- **与狗男联动**：料理每次给玛露额外+50能量；玛露获得"忠犬"+40%攻击力
- **微调经验**：见诊断规则库 NC-003（需全队最高战斗力，必要时让其他英雄脱装备）

### 格罗克 / Gorok (大虫子) - 战士
- **伤害类型**：物理
- **最低投资要求（F3关键原因）**：3红效果"友方对宿敌伤害+35%"，宿敌系统让全队专注集火同一目标，3红是其核心辅助输出价值；无3红时仅为普通前排坦克
- **宿敌机制**：对友方威胁最大（最深入友方区域）的敌人；死亡后改为最高攻击力敌人
- **终极释放条件**：需要满能量+角甲虫满3层饱食（节奏较慢，急速影响触发频率）
- **终极效果**：全队护盾=80%格罗克最大生命值+每0.5秒回4%生命值，是关键生存技能

### 菲泰拉 / Veithael (黄毛) - 坦克
- **伤害类型**：物理
- **急速35最优原因**：被动不被控时每秒+50能量，技能释放节奏在急速35时达到能量爆发频率×减伤积累×队伍技能轴同步的综合最优点；过高过低都影响节奏
- **核心价值（SI+30）**：全队分享"每1%最大生命值加成=+1%攻击力+3闪避"，是坦克型增益辅助
- **被控风险**：被控时每秒-150能量 + 清除"昂首"所有层数，需要队伍提供免控支持
- **微调经验**：见诊断规则库 NC-002（Haste 35）

### 车惠怡 / Cha Hae-in (车女) - 战士
- **伤害类型**：物理 + 神圣伤害
- **梦魇核心地位原因**：① Final Boss标记全队集火时额外输出（每1.5秒+200-300%攻击力），② 终极AOE伤害适合清小队，③ 压制debuff（-90命中/-60魔法效率）极强
- **E30关键**：Final Boss死亡后-20%攻防（最多叠3次=-60%），BOSS战中战斗力极高
- **高投资需要原因**：E60"伤害必定命中"+每20暴击率增加40%神圣伤害（上限160%），需要高刻印解锁全部输出潜力
- **与君王联动**：配合程肖宇(君王)时获得75%的君王升级效果
