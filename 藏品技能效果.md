## 1. 赏金 / Bounty Hunter's Decree (辅助) — 雷恩 (Raine)

**技能名**：Burning Target（锁定悬赏）

| 等级 | 效果 |
|------|------|
| 一星 (Lv.0) | 进入战场后，对最近的敌方英雄施加悬赏标记15秒，使其受到的伤害增加10%，最高增加40% |
| 二星 (Lv.1) | 友方对被悬赏敌人造成伤害时，获得20点暴击率+15点暴伤增幅，并回复持有者ATK 120%的HP（每3秒1次） |
| 三星 (Lv.2) | 悬赏额外伤害提升至20% |

> **诊断要点**：赏金+2 (Lv.1) 提供永久 CRIT/CDA buff，是 CR 中最重要的藏品之一。一星和二星差距极大。

## 2. 魅魔 / Captive Wings (法师) — 梅希拉 (Mehira)

**技能名**：Beguiling Whispers（蛊惑低语）

| 等级 | 效果 |
|------|------|
| 一星 (Lv.0) | 大招结束后蛊惑最近敌人2秒，被蛊惑者只能普攻 |
| 二星 (Lv.1) | 同时召唤仆从持续攻击，造成ATK 200%伤害并吸血100%，最多1个仆从 |
| 三星 (Lv.2) | 受到超过最大HP 15%的单次伤害时，牺牲仆从抵消伤害 |

## 3. 牛头 / Horn of Dominion (坦克) — 阿诺奇 (Anoki)

**技能名**：Battlecry Resounds（战号回荡）

| 等级 | 效果 |
|------|------|
| 一星 (Lv.0) | 进入战斗后全体获得护盾，抵消持有者最大HP 12%的伤害，持续6秒 |
| 二星 (Lv.1) | 战斗中每15秒额外释放1次，最多3次 |
| 三星 (Lv.2) | 为附近友方提供15点伤害抗性+20点韧性 |

> **诊断要点**：Reddit 提到 SP女骑 (Estrilda) 在诅咒梦境中需要 Lv.3 牛头，因为队伍容易过热 (overheat)。

## 12. 雷神 / Stormwrath's Lull (法师) — 扎弗雷尔 (Zaphrael)

**技能名**：Stormy Retribution（风暴报应）

| 等级 | 效果 |
|------|------|
| 一星 (Lv.0) | 伤害使敌人带电4秒（-15伤害抗性，不叠加）；每3次伤害触发闪电，造成ATK 250% AoE伤害（每6秒1次） |
| 二星 (Lv.1) | 每1个带电敌人，持有者伤害+7%（最高21%） |
| 三星 (Lv.2) | 攻击被控制的带电敌人，ATK额外+60% |

> **诊断要点**：雷神 Lv.2/3 是核心输出藏品，Reddit 频繁推荐放在秋神 (Aurelia) 和夏神 (Daemia) 等法师身上。

## 14. 时钟 / Timeline Watch (坦克) — 奥瑟罗斯 (Orthos)

**技能名**：Time Control（时间控制）

| 等级 | 效果 |
|------|------|
| 一星 (Lv.0) | 首次受伤时停止时间3秒，只有持有者可行动（后续友方触发时持续时间-40%） |
